# マイプランナー

スマートフォン向けの自己管理・知識ベースアプリです。PWA（Progressive Web App）として動作し、オフラインでも利用できます。

## 機能

- **ホーム** — 今日のサマリーを確認
- **カレンダー** — 予定を管理
- **タスク** — やることリストを管理
- **目標** — 長期目標を設定・追跡
- **設定** — テーマなどのカスタマイズ

## 使い方

### ローカルで起動する

`start.bat` をダブルクリックするか、HTTPサーバーで配信してください。

```bash
npx serve .
```

その後ブラウザで `http://localhost:3000` を開きます。

### PWAとしてインストールする

スマートフォンのブラウザでアクセスし、「ホーム画面に追加」を選択するとアプリとして使用できます。

## 技術スタック

- Vanilla JavaScript (ES Modules)
- Service Worker によるオフラインキャッシュ
- KaTeX（数式レンダリング）
- LocalStorage によるデータ保存
